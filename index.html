HTML

  • The head element contains information about the webpage:
    • The meta tag contains meta data for generating the webpage, such as charset, http-equiv, content, and name.
    • The title element renders in the internet tab and is important for SEO.
  • The body element represents the visible content shown to the user:
    • The header element is at the top of the page
    • The main element comes after the header element contains sections, headers and more
  • Other important tags include class and id

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • ".class" allows you to use class selectors.
  • learnlayout.com is incredibly useful as a refresher for css layout!
  • htmlcheatsheet.com/css/ seems helpful for previewing css quickly

GitHub

  • git status: checks what branch we are currently on
  • git checkout main: goes to the main branch
  • git pull origin main: ensures that we are working on the most updated version of the repo
  • git checkout -b branch-name: creates a new branch and switches to it
  • . code: opens VS Code

JavaScript

  • A variable is a named container that allows us to store data in our code using "var"
  • Control flow is the order in which a computer executes code in a script.
  • Conditionals syntax:
    if (condition) {
    // block of code to be executed if condition is true
    } else {
    //different code to be executed if condition is false
    }
